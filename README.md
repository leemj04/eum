## e:um 
청년과 정책을 이어주는 AI 기반 맞춤형 청년정책 추천 및 인생 로드맵 설계 서비스


<br />

 
## 🚀 서비스 소개

**e:um (이음)**  
청년과 정책을 이어주는 **AI 기반 맞춤형 정책 추천 및 인생 로드맵 설계 서비스**입니다.  
분산된 청년 정책 정보를 통합하고, AI가 개인 맞춤형으로 정책과 혜택을 자동 추천하여  
청년의 경제적·사회적 기회를 확장하는 것을 목표로 합니다.



<br />

## ✨ 핵심 기능

### 🧠 1. AI 대화형 정책 추천 (LangGraph 기반)
- LangGraph로 구성된 대화 흐름을 통해 자연스럽게 조건 파악  
- 나이, 지역, 학력, 소득, 관심사 기반 맞춤형 탐색 및 추천  
- 실시간 정책 필터링 + 적합도 점수 기반 정렬

### 🗺️ 2. 개인 로드맵 자동 생성
- 시기별 신청 가능 정책을 자동 정리  
- 예: "3월 청년수당 → 5월 취업성공패키지 → 7월 주거지원금"  
- 드래그&드롭 가능한 블록 조립식 UI 제공

### 📅 3. 알림 & 캘린더 연동
- Flutter Local Notification 기반 푸시 알림  
- 신청 마감일, 제출일 자동 캘린더 등록  
- D-7 / D-3 / D-1 알림 제공

### 🔎 4. 정책 탐색 페이지
- 필터(지역·나이·소득·분야 등) 기반 정책 검색  
- 보기 쉬운 정책 카드형 UI

### 📊 5. 나만의 대시보드
- 프로필  
- 추천 정책  
- 현재 신청 현황  
- 최신 혜택 요약  
한 화면에서 한눈에 확인 가능
